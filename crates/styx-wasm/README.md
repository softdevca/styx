# styx-wasm

[![crates.io](https://img.shields.io/crates/v/styx-wasm.svg)](https://crates.io/crates/styx-wasm)
[![documentation](https://docs.rs/styx-wasm/badge.svg)](https://docs.rs/styx-wasm)
[![MIT/Apache-2.0 licensed](https://img.shields.io/crates/l/styx-wasm.svg)](./LICENSE)

WebAssembly bindings for the [Styx](https://github.com/bearcove/styx) parser. Enables parsing and validating Styx documents in browsers and other WASM runtimes.
